# ADR-0001: Project Direction

Date:
2026-6-28

## Decision

Helios Vision will prioritize on-device AI architecture
targeting Android smartphones.

## Reason

Primary goals:

- Protect user privacy
- Avoid mandatory cloud dependency
- Enable offline usage
- Support users in environments with unreliable connectivity

## Consequences

Positive:

- Better privacy
- Lower latency
- More control

Challenges:

- Model optimization required
- Mobile hardware constraints
- Battery considerations
